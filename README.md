# PETLY 🐾

**PETLY** — this is a volunteer project that helps animals find homes and encourages people to do good deeds.

Site: [mikhaildadaev.github.io/petly](https://mikhaildadaev.github.io/petly/)

---

## 📖 About project

PETLY — this is a platform for animal volunteers, created by volunteers for volunteers. We believe that every pet deserves a home, and every person can be a part of a good cause.

### What We Do:
- 🏠 Help shelter animals find families
- 🤝 Unite volunteers who are ready to care
- 💛 Make kindness simple and accessible

## 🛠️ Installation and Running

```bash
# Clone the repository
git clone https://github.com/mikhaildadaev/petly.git
# Navigate to the project folder
cd petly
# Install dependencies
npm install
# Run in development mode
npm run docs:dev
# Build the project
npm run docs:build
# Preview the built project
npm run docs:preview
```
